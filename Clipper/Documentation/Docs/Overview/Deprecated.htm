Deprecated

Depricated functions encapsulate calls to new and improved functions or classes and will be removed in a future update. The precompiler directive 'use_deprecated' allows users to easily locate and enable/disable these functions. (It's enabled by default.)

All structures and functions that were labelled 'deprecated' in Clipper version 6.0 have now been removed.

Deprecated functions:

OffsetPaths Replaced by the ClipperOffset class.
